###Usage
**How to run SQL script**
Suppose SQL script is stored in /path/to/sql/setup.sql
1. Run mysql server
2. Enter the client, e.g.
```shell
mysql -u <Username> -p
```
3. Run the SQL script
```
source /path/to/sql/setup.sql
```
